# Projeto Startrr

Este projeto é uma aplicação web desenvolvida para apresentar uma vitrine de aulas, suporte ao usuário e outras funcionalidades relacionadas ao ensino online.

## Estrutura do Projeto

- **index.html**: Página inicial do projeto.
- **aulas.html**: Página dedicada à listagem de aulas.
- **suporte.html**: Página de suporte ao usuário.
- **vitrine.html**: Página de vitrine de produtos/aulas.
- **script.js**: Scripts JavaScript para funcionalidades dinâmicas.
- **styles.css**: Estilos CSS para o layout e design do site.
- **assets/img/aulas/**: Imagens relacionadas às aulas.
- **specs/**: Especificações técnicas e arquivos auxiliares.

## Como executar

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em seu navegador.
3. Navegue pelas páginas conforme necessário.

## Funcionalidades

- Simulador financeiro de MEI (cálculos baseados em ações do empreendedor sob seu negócio)
- Visualização de aulas disponíveis
- Página de suporte ao usuário
- Vitrine de produtos/aulas
- Layout responsivo

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
